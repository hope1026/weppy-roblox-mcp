## Installation

### 1. Install Roblox Plugin

Download the plugin file and install it manually:

1. **Download** `WeppyRobloxMCP.rbxm` from [Releases](https://github.com/hope1026/roblox-mcp/releases)
2. **Open** Roblox Studio
3. **Go to** Plugins tab → **Plugins Folder**
4. **Copy** the downloaded `.rbxm` file into the opened folder
5. **Restart** Roblox Studio
6. The plugin will appear in the Plugins tab

### 2. Configure MCP Server

Works with any AI app that supports MCP. Choose your AI app:

| AI App | Setup Guide |
|--------|-------------|
| Claude Code | [Setup](installation/claude-code.md) |
| Claude Desktop | [Setup](installation/claude-app.md) |
| Codex CLI | [Setup](installation/codex-cli.md) |
| Codex Desktop | [Setup](installation/codex-app.md) |
| Gemini CLI | [Setup](installation/gemini-cli.md) |
| Cursor, Windsurf, etc. | [Setup](installation/antigravity.md) |

### 3. Connect

1. Launch Roblox Studio
2. Go to Plugins tab → Click **W-MCP**
3. Click **Connect** button
4. Confirm "Connected" status
5. Start developing with AI!
