# Roblox MCP 機能

AIとRoblox Studioを接続して、さまざまな作業を自動化できます。カテゴリ別にどんなことができるかを紹介します。

---

## Basic（無料）

### オブジェクト管理

パーツ、モデル、フォルダなどのゲームオブジェクトを作成、複製、削除、移動できます。「赤いパーツを作って」「このモデルを複製して横に置いて」のような作業が可能です。

### プロパティ設定

オブジェクトのサイズ、色、位置、マテリアルなどすべてのプロパティを読み取り、変更できます。複数のプロパティを一度に変更したり、現在の値を基準に相対的に調整することも可能です。

### スクリプト

新しいスクリプトを作成し、既存のコードを読み取り、修正できます。特定の行だけを編集したり、スクリプト内のテキストを検索したり、どのモジュールを参照しているか依存関係を把握することも可能です。

### セレクション

現在選択しているオブジェクトを確認し、選択を変更し、選択をクリアできます。Studioで作業中の対象を理解し制御するために不可欠です。

### 検索

名前やクラス（種類）でゲーム内のオブジェクトを検索できます。

### タグ

オブジェクトにタグを付けたり外したり、特定のタグが付いたオブジェクトを一度に検索できます。同じ種類のオブジェクトをグループで管理するときに便利です。

### カメラ

カメラを特定のオブジェクトや位置に移動できます。作業中のエリアを素早く確認するのに便利です。

### ログ

ゲーム実行中に発生した出力ログとエラーを確認できます。バグの発見と修正に役立ちます。

### 接続

MCPサーバーとRoblox Studioの接続状態と使用量を確認できます。

---

## Pro

より強力な機能が必要なら、Proにアップグレードできます。

### 高度なセレクション

選択したオブジェクトのプロパティやソースコードを含む詳細なコンテキストを確認し、選択階層を探索し、選択にアイテムを追加・削除し、Place情報やゲームサービスを確認できます。高度なワークフローや自動化に便利です。

### 高度なスクリプト

複数のスクリプトの内容を一括で置換したり、Luauコードを即座に実行できます。繰り返しのコード修正や素早いテストに便利です。

### 高度な検索

ファイルツリー全体を探索したり、プロパティやタグでオブジェクトを検索できます。プロジェクト構造を一目で把握し、特定のオブジェクトのすべての子孫や祖先を確認できます。

### 一括操作

数百のオブジェクトを一度に作成、削除、修正、複製できます。マップに木を植えたり、すべてのパーツの色を変えたり、パターンに従ってオブジェクトを配置する大規模な作業を素早く処理します。

### アセット

Roblox Creator Storeから無料モデルを検索してすぐに挿入できます。「中世の城モデルを探して」「武器モデルを入れて」のように会話だけでアセットを追加します。

### 環境設定

照明、空、霧、地形プロパティ、時間帯を自然言語で設定できます。「夕焼けの雰囲気にして」「深夜12時に設定して」のようなリクエストが可能です。

### 状態管理

Workspaceの現在の状態をスナップショットとして保存し、最近の変更やビューポートの状態を確認できます。複雑な作業中に現在の状態を把握するのに役立ちます。

### 空間分析

オブジェクトのサイズと位置を正確に把握し、空きスペースを見つけ、衝突の有無を確認できます。オブジェクトをグリッドに揃えることも可能です。

### レイキャスト

目に見えない光線を飛ばして地形やオブジェクトとの衝突地点を見つけます。地面の位置の検出、オブジェクト配置の可否確認、スポーン位置の検索、移動可能エリアの分析など、ゲームレベルデザインに不可欠な機能です。

### 地形

ブロック、球、円柱、くさびなどさまざまな形状で地形を埋めたり、削ったり、滑らかにできます。マテリアルや色を変更したり、山や川のある自然の地形を手続き的に自動生成することもできます。

### 可視化

エリア、マーカー、ラインなどを画面に表示して、作業中の空間を視覚的に確認できます。デバッグやレベルデザインのレビューに便利です。
