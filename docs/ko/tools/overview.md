# Roblox MCP 기능

AI와 Roblox Studio를 연결해서 다양한 작업을 자동화할 수 있습니다. 아래는 카테고리별로 어떤 일을 할 수 있는지 소개합니다.

---

## Basic (무료) — 68개 도구

### 오브젝트 관리 — 13개

파트, 모델, 폴더 등 게임 오브젝트를 만들고, 복제하고, 삭제하고, 이동할 수 있습니다. "빨간 파트를 만들어줘", "이 모델을 복제해서 옆에 놔줘" 같은 작업이 가능합니다.

### 속성 설정 — 10개

오브젝트의 크기, 색상, 위치, 재질 등 모든 속성을 읽고 변경할 수 있습니다. 여러 속성을 한번에 바꾸거나, 현재 값을 기준으로 상대적으로 조정하는 것도 가능합니다.

### 스크립트 — 9개

새 스크립트를 만들고, 기존 코드를 읽고, 수정할 수 있습니다. 특정 줄만 편집하거나, 스크립트 안에서 텍스트를 검색하거나, 어떤 모듈을 참조하는지 의존성을 파악하는 것도 가능합니다.

### Studio 제어 — 12개

현재 선택한 오브젝트를 확인하고, 선택을 변경하고, Studio 명령을 실행할 수 있습니다. Place 정보나 게임에 사용되는 서비스 목록도 조회할 수 있습니다.

### 검색 — 2개

이름이나 클래스(종류)로 게임 안의 오브젝트를 찾을 수 있습니다.

### 태그 — 5개

오브젝트에 태그를 붙이거나 떼고, 특정 태그가 달린 오브젝트를 한번에 찾을 수 있습니다. 같은 종류의 오브젝트를 그룹으로 관리할 때 유용합니다.

### 카메라 — 4개

카메라를 특정 오브젝트나 위치로 이동시킬 수 있습니다. 작업 중인 영역을 빠르게 확인할 때 편리합니다.

### 로그 — 3개

게임 실행 중 발생한 출력 로그와 에러를 확인할 수 있습니다. 버그를 찾고 수정하는 데 도움이 됩니다.

### 연결 — 2개

MCP 서버와 Roblox Studio의 연결 상태와 사용량을 확인할 수 있습니다.

---

## Pro — 72개 추가 도구

더 강력한 기능이 필요하다면 Pro로 업그레이드할 수 있습니다.

### 고급 스크립트 — 2개

여러 스크립트의 내용을 한번에 일괄 치환하거나, Luau 코드를 즉시 실행할 수 있습니다. 반복적인 코드 수정이나 빠른 테스트에 유용합니다.

### 고급 검색 — 6개

파일 트리 전체를 탐색하거나, 속성이나 태그로 오브젝트를 검색할 수 있습니다. 프로젝트 구조를 한눈에 파악하고, 특정 오브젝트의 모든 자손이나 조상을 조회할 수 있습니다.

### 대량 작업 — 8개

수백 개의 오브젝트를 한번에 생성, 삭제, 수정, 복제할 수 있습니다. 맵에 나무를 심거나, 모든 파트의 색을 바꾸거나, 패턴에 따라 오브젝트를 배치하는 대규모 작업을 빠르게 처리합니다.

### 에셋 — 7개

Roblox Creator Store에서 무료 모델을 검색하고 바로 삽입할 수 있습니다. "중세 성 모델 찾아줘", "무기 모델 넣어줘"처럼 대화만으로 에셋을 추가합니다.

### 환경 설정 — 5개

조명, 하늘, 안개, 지형 속성, 시간대를 자연어로 설정할 수 있습니다. "석양 분위기로 바꿔줘", "밤 12시로 설정해줘" 같은 요청이 가능합니다.

### 상태 관리 — 9개

Workspace의 현재 상태를 스냅샷으로 저장하고, 변경 이력을 추적할 수 있습니다. 복잡한 작업을 할 때 현재 상태를 파악하고 관리하는 데 도움이 됩니다.

### 공간 분석 — 5개

오브젝트의 크기와 위치를 정확하게 파악하고, 빈 공간을 찾고, 충돌 여부를 확인할 수 있습니다. 오브젝트를 그리드에 맞춰 정렬하는 것도 가능합니다.

### 레이캐스트 — 8개

보이지 않는 광선을 쏴서 지형이나 오브젝트와의 충돌 지점을 찾습니다. 지면 위치 찾기, 오브젝트 배치 가능 여부 확인, 스폰 위치 찾기, 이동 가능 영역 분석 등 게임 레벨 디자인에 필수적인 기능입니다.

### 지형 — 14개

블록, 구, 원통, 쐐기 등 다양한 형태로 지형을 채우고, 깎고, 부드럽게 다듬을 수 있습니다. 재질과 색상을 변경하거나, 산과 강이 있는 자연 지형을 절차적으로 자동 생성할 수도 있습니다.

### 시각화 — 7개

영역, 마커, 선 등을 화면에 표시하여 작업 중인 공간을 시각적으로 확인할 수 있습니다. 디버깅이나 레벨 디자인 검토에 유용합니다.
