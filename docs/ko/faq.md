# 자주 묻는 질문 (FAQ)

## 설치 관련

### Q: 어떤 AI 앱을 사용해야 하나요?

MCP를 지원하는 모든 앱에서 사용할 수 있습니다:

| 앱 | 추천 대상 |
|----|----------|
| **Claude Desktop** | 일반 사용자, 초보자 |
| **Claude Code** | 터미널 선호, 개발자 |
| **Cursor** | 코드 에디터와 함께 사용 |
| **Codex CLI** | OpenAI 선호 사용자 |
| **Gemini CLI** | Google 선호 사용자 |

### Q: Node.js가 꼭 필요한가요?

네, MCP 서버 실행에 Node.js 18.0 이상이 필요합니다.

설치 확인:
```bash
node --version
```

### Q: 플러그인은 무료인가요?

- **무료 버전**: 68개 기본 도구 무제한 사용
- **Pro 버전**: 140개 전체 도구 무제한 사용

무료 사용자도 Pro 도구를 일일 할당량 내에서 사용할 수 있습니다.

## 연결 문제

### Q: "Not Connected" 상태가 지속됩니다

1. **MCP 서버 실행 확인**
   - AI 앱에서 MCP 서버가 정상 실행 중인지 확인
   - 터미널에서 수동 실행 테스트:
     ```bash
     npx weppy-roblox-mcp
     ```

2. **방화벽 설정**
   - 포트 3002가 차단되지 않았는지 확인
   - Windows: Windows Defender 방화벽 → 인바운드 규칙
   - macOS: 시스템 환경설정 → 보안

3. **플러그인 재시작**
   - Disconnect → 잠시 대기 → Connect

### Q: 연결되었는데 명령이 안 됩니다

1. AI 앱에서 Roblox MCP가 활성화되었는지 확인
2. "Roblox Studio에서 파트 만들어줘"처럼 명확하게 요청
3. Studio Output 창에서 에러 메시지 확인

### Q: 두 개의 AI 앱에서 동시에 사용할 수 있나요?

아니요, MCP 서버는 한 번에 하나의 클라이언트만 연결할 수 있습니다.

## 사용 관련

### Q: AI가 원하는 대로 안 합니다

명확하고 구체적으로 요청하세요:

**애매한 요청:**
```
파트 만들어줘
```

**좋은 요청:**
```
(10, 5, 0) 위치에 크기 5x1x5의 빨간색 파트 만들어줘
```

### Q: 스크립트가 제대로 작동하지 않습니다

1. AI에게 스크립트 내용 확인 요청:
   ```
   이 스크립트 내용 보여줘
   ```

2. 에러 확인:
   ```
   최근 에러 보여줘
   ```

3. 단계별로 디버깅 요청

### Q: 대량 작업이 안 됩니다

무료 사용자는 대량 작업에 제한이 있습니다:
- 한 번에 생성/수정: 최대 10개
- 일일 대량 작업 횟수: 10회

더 많은 작업이 필요하면:
- 여러 번 나눠서 요청
- Pro 구독 고려

### Q: 에셋 검색이 안 됩니다

1. Creator Store 검색은 Pro 도구입니다 (무료: 일일 5회)
2. 검색어를 영어로 시도해보세요
3. 구체적인 검색어 사용:
   ```
   "low poly tree" 무료 모델 검색해줘
   ```

## 보안 관련

### Q: 외부에서 접근할 수 있나요?

아니요, MCP 서버는 `localhost(127.0.0.1)`에서만 실행됩니다. 외부 네트워크에서 접근할 수 없습니다.

### Q: 어떤 데이터가 전송되나요?

- AI 앱 → MCP 서버: 실행할 명령 (로컬)
- MCP 서버 → Studio 플러그인: 명령 데이터 (로컬)
- 외부 전송: 없음 (에셋 검색 제외)

### Q: 보호되는 항목이 있나요?

다음 경로는 접근이 차단됩니다:
- CoreGui
- CorePackages
- RobloxScript
- 기타 시스템 경로

## Pro 관련

### Q: Pro와 무료의 차이점은?

| 기능 | 무료 | Pro |
|-----|:----:|:----:|
| 기본 도구 (68개) | ✅ 무제한 | ✅ 무제한 |
| Pro 도구 (72개) | 일일 할당량 | ✅ 무제한 |
| 대량 작업 | 10개/회, 10회/일 | 무제한 |
| 에셋 검색 | 5회/일 | 무제한 |
| 지형 생성 | 3회/일 | 무제한 |

### Q: 할당량이 리셋되는 시간은?

매일 자정(UTC 기준)에 리셋됩니다.

### Q: Pro 구독은 어떻게 하나요?

Creator Store에서 Pro 플러그인을 구매하세요:
- [Pro 플러그인 구매](https://create.roblox.com/store/asset/YOUR_PRO_ASSET_ID)
